## Introduction

### Overview
Cavrnus Spatial Connector provides tools to add multi-user collaboration, real-time communication, and an editable 3D environment in a persistent virtual space. The tool enhances 3D workflows by offering a seamless drag-and-drop toolkit for embedding collaborative features directly into Unity projects.

➡️ Learn more at [www.cavrnus.com](https://www.cavrnus.com/)

[![Discord](https://img.shields.io/badge/Cavrnus_Discord-Join_Now-5865F2?logo=discord&logoColor=white&style=flat)](https://discord.gg/AzgenDT7Ez) 

Ask questions, share feedback, or get support directly from the team!

## Features

- **Persistence**  
  All interactions are time-stamped and logged in the Journal, enabling full session history and analytics via API.

- **Multi-User Copresence**  
  Real-time data sync across users and platforms for collaborative experiences.

- **Robust Voice & Video**  
  Built-in low-latency voice, video, and screen sharing powered by WebRTC.

- **Flexible Deployment**  
  Use Cavrnus via our hosted cloud, Virtual Private Cloud, or on-premises options.

- **Quick Start & Extensibility**  
  Drop-in integration with existing projects; fully customizable with pre-built and extensible components.

## Installation

To install the Cavrnus Spatial Connector using Unity's Package Manager:

### Step 1: Add Scoped Registry

1. In Unity, open **Edit > Preferences > Package Manager**   _(macOS: Unity > Preferences > Package Manager)_
2. Under **Scoped Registries**, click **Add**.
3. Fill in the following:

   - **Name:** Cavrnus  
   - **URL:** `https://packages.cavrn.us`  
   - **Scope(s):** `com.cavrnus`

4. Save the changes.

### Step 2: Add the Package

1. In Unity, open **Window > Package Manager**.
2. In the **Package Manager**, click the dropdown in the top-left corner and select **My Registries**.
3. Find **Cavrnus Spatial Connector** in the list.
4. Click **Install**.

> ℹ️ If the package does not appear, try restarting Unity or verifying that the scoped registry was added correctly.

> ℹ️ In some versions of Unity, the My Registries section may not appear. If this happens, please use Method 2: Git Deployment instead.
